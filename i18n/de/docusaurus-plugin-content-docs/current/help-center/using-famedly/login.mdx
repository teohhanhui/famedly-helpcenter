## Die Anmeldung

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie die Famedly Web App.</li>
    <li>Geben Sie die URL Ihres Matrix Servers in das Eingabefeld <b>Anmelde-URL</b> ein.</li>
    <li>Klicken Sie auf <b>Weiter</b>.</li>
    <li>Geben Sie Ihren Benutzernamen in das Eingabefeld <b>Benutzername</b> und Ihr Passwort in das Eingabefeld <b>Passwort</b> ein.</li>
    <li>Klicken Sie auf <b>Anmelden</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>

    <li>Tippen Sie auf <b>Weiter</b>.</li>
    <li>Geben Sie Ihren Benutzernamen in das Eingabefeld <b>Benutzername</b> und Ihr Passwort in das Eingabefeld <b>Passwort</b> ein.</li>

## Accountweite Einstellungen beim Login

Nach dem Login werden Sie zu einer Seite mit verschiedenen Schaltern für <b>Datenschutzeinstellungen</b> (Chat-Features) und für das <b>Benutzerverzeichnis</b> weitergeleitet.  

Eine Beschreibung zu den Chat-Features finden Sie hier: [Datenschutzeinstellungen](../your-account/account-settings#datenschutz)\
Informationen zur Einstellung zum Benutzerverzeichnis finden Sie hier: [Verzeichnis](../your-account/account-settings#Verzeichnis)

Falls Sie bereits das [Chat-Backup](../first-steps/first-login#chat-backup-aktivieren) aktiviert haben, werden Sie nun zum Verifikationsprozess weitergeleitet, wenn Sie auf <b>Fortfahren</b> klicken. 

## Verifizieren Ihres Chat-Backups

Bevor Sie alle Ihre Nachrichten lesen können, müssen Sie Ihren Account verifizieren. Alle Informationen dazu finden Sie unter [QR-Verifikation](../using-famedly/qr-verification). 
Falls Sie aus irgendeinem Grund keine QR-Codes scannen können, gibt es alternativ die Möglichkeit, Nummern zu vergleichen. Dies wird ebenfalls unter [QR-Verifikation](../using-famedly/qr-verification) beschrieben.
