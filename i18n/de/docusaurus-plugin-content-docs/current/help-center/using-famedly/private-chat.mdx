## Privatchats starten


Um Privatchats mit einer Person zu starten, folgen Sie den folgenden Schritten.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicke auf den <b>Chat starten</b> Button in der oberen linken Ecke der App.</li>
    <li>Klicken Sie auf <b>Chat starten</b>.</li>
    <li>Klicken Sie auf den Namen der Person, mit der Sie einen Privatchat starten möchten.</li>
    <li>Klicken Sie auf <b>Chat starten</b> oder <b>Chat öffnen</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippen Sie auf <b>Chats</b> am unteren Bildschirmrand.</li>
    <li>Tippen Sie auf die <b>+ Neu</b> Schaltfläche unten rechts auf Ihrem Bildschirm.</li>
    <li>Tippen Sie auf den Namen der Person, mit der Sie einen Privatchat starten möchten.</li>
    <li>Tippen Sie auf das <b>Sprechblasen-Symbol</b>.</li>
  </ol>
  </TabItem>
</Tabs>

:::info

Wenn Sie nicht die Person finden, mit der Sie einen privaten Chat starten möchten, schauen Sie bitte hier nach.

:::

## Privatchats beitreten

Um Privatchats beitreten zu können, muss eine Person zunächst einen Privatchat mit Ihnen gestartet haben. Wenn die andere Person einen privaten Chat mit Ihnen begonnen hat, wartet eine Einladung zum Beitritt auf Sie.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Sie können einer Privatchats nur beitreten, wenn Sie eine Einladung erhalten haben.</li>
    <li>Alle Einladungen befinden sich am oberen Ende der Chats Liste.</li>
    <li>Klicken Sie auf die Privatchats zu denen Sie eingeladen wurden.</li>
    <li>Akzeptieren Sie die Einladung.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Sie können einer Privatchats nur beitreten, wenn Sie eine Einladung erhalten haben.</li>
    <li>Alle Einladungen befinden sich am oberen Ende der Chats Liste.</li>
    <li>Klicken Sie auf die Privatchats zu denen Sie eingeladen wurden.</li>
    <li>Akzeptieren Sie die Einladung.</li>
  </ol>
  </TabItem>
</Tabs>

:::danger

Wenn Sie die Einladung ablehnen, müssen Sie die Person erneut bitten, einen Privatchat mit Ihnen zu beginnen oder Sie können den Privatchat starten.

:::

## Privatchats verlassen

Sie haben immer die Möglichkeit, einen privaten Chat zu verlassen. Bitte beachten Sie, dass Sie einen neuen Privatchat beginnen müssen, um das Gespräch mit der anderen Person fortzusetzen.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf <b>ℹ</b> in der rechten oberen Ecke des Bildschirms eines Privatchats, um die Details zu öffnen.</li>
    <li>Klicken Sie auf <b>Gespräch verlassen</b>.</li>
    <li>Klicken Sie auf <b>Beenden</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippe auf den Kopf eines privaten Chats, um die Details des privaten Chats zu öffnen.</li>
    <li>Tippe <b>Chat verlassen</b> unten auf der Seite.</li>
    <li>Wählen Sie <b>Ja</b>.</li>
  </ol>
  </TabItem>
</Tabs>

:::note

Mit dem Verlassen des Privaten Chats wird die Unterhaltung für Sie archiviert. Sie können das Archiv in den Einstellungen finden und somit ältere Inhalte nachlesen.

:::

## Privatchats löschen

Um einen privaten Chat zu löschen, müssen Sie zuerst den Raum verlassen. Danach folgen Sie den Schritten unten, um den privaten Chat zu löschen. Sie werden den privaten Chat immer für sich selbst löschen - die andere Person hat noch Zugriff auf die vergangenen Gespräche.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Tippen Sie rechts neben dem Filterfeld auf Ihr <b>Profilbild oder Namens Initialen</b> um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf <b>Archiv</b>.</li>
    <li>Klicken Sie auf das <b>Haken-Symbol</b> in der rechten oberen Ecke des Bildschirms.</li>
    <li>Wählen Sie einen oder mehrere private Chats, die Sie löschen möchten.</li>
    <li>Klicken Sie auf das <b>Papierkorb-Symbol</b>.</li>
    <li>Klicken Sie <b>Ja.</b></li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippen Sie rechts neben dem Filterfeld auf Ihr <b>Profilbild oder Namens Initialen</b> um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie <b>Archiv</b>.</li>
    <li>Tippen Sie <b>Archiv leeren</b>.</li>
    <li>Tippen Sie <b>Löschen</b> um alle privaten Chats & Gruppen zu entfernen.</li>
  </ol>
  </TabItem>
</Tabs>

:::note

Das Laden des Archivs kann einige Zeit in Anspruch nehmen.

:::

## Privatchats stummschalten

Wenn du nicht durch einen bestimmten privaten Chat gestört werden möchtest, kannst du ihn stummschalten.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf <b>ℹ</b> in der rechten oberen Ecke des Bildschirms eines Privatchats, um die Details zu öffnen.</li>
    <li>Klicke auf das <b>Glockensymbol</b> um den Chat stumm zuschalten.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippe auf die Kopfzeile eines Privatchats, um die Details zu öffnen.</li>
    <li>Tippen Sie auf das <b>Glockensymbol</b> um den Chat stumm zuschalten.</li>
  </ol>
  </TabItem>
</Tabs>

:::note

Sie können die Benachrichtigungen eines Chats wieder aktivieren, indem Sie den selben Schritten folgen und auf das durchgestrichene **Glockensymbol** klicken.

:::
