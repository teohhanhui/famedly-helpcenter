# Share files in Famedly


<Tabs> <TabItem value="desktop" label="Desktop" default>
1. Open a group or private chat you want to share the file with.
2. Click on the **Paperclip icon**.
3. Select the file you want to share and confirm.</TabItem> <TabItem value="mobile" label="Mobile">
1. Open a group or private chat you want to share the file with.
2. Click on the **Paperclip icon**.
3. Select **Document**.
4. Select the file you want to share and confirm.</TabItem> </Tabs>

# Share images and videos


<Tabs> <TabItem value="desktop" label="Desktop" default>
1. Open a group or private chat you want to share the picture or video with.
2. Click on the **paperclip icon**.
3. Select the image/video from your file system.
4. Click **Yes** to confirm sharing..</TabItem> <TabItem value="mobile" label="Mobile">
1. Open a group or private chat you want to share the picture or video with.
2. Tap **Video**, **Gallery** or **Camera**.
3. Select a picture/video or take a picture/video with the camera.
4. Optional: Tap **Black icon** if you want to blacken parts of the picture.
5. Tap the **Hook icon**.</TabItem> </Tabs>

# Download files and pictures

You are able to download files and pictures that you received in a chat.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Open the chat with the file you want to download.</li>
    <li>Click on the file, then on the download arrow in the top right corner.</li>
    <li>Confirm the notification.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Open the chat with the file you want to download.</li>
    <li>Click on the file an then on the arrow in the top right corner.</li>
    <li>Confirm the notification.</li>
  </ol>
  </TabItem>
</Tabs>

# Make image sections unrecognizable


<Tabs> <TabItem value="desktop" label="Desktop" default>
1. This feature is only available on mobile till now.</TabItem> <TabItem value="mobile" label="Mobile">
1. Follow the steps of sharing an image.
2. Before tapping the **Hook icon** you can select the colors black or white by clicking on the small painter scroll icon in the bottom right corner of the app when you have selected an picture.
3. Tap the **Hook icon**.</TabItem> </Tabs>
