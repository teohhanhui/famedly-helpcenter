## Dateien in Famedly teilen

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie eine Gruppe oder einen Privatchat, mit dem Sie die Datei teilen möchten.</li>
    <li>Klicken Sie auf das <b>Büroklammer-Symbol</b>.</li>
    <li>Wählen Sie die Datei aus, die Sie freigeben möchten und bestätigen Sie.</li>
  </ol>

:::info

Sie können Dokumente auch in den Chat ziehen, wo Sie sie verschicken möchten.

:::

:::info

Kopierte dateien können auch direkt in den Chat eingefügt werden, entweder durch das Rechtsklickmenü in der Nachrichtenzeile oder durch ein Tastenkürzel. Unter Firefox kann nur eine Datei gleichzeitig eingefügt werden.

:::

  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Öffnen Sie eine Gruppe oder einen Privatchat, mit dem Sie die Datei teilen möchten.</li>
    <li>Klicken Sie auf das <b>Büroklammer-Symbol</b>.</li>
    <li>Wählen Sie <b>Dokument</b>.</li>
    <li>Wählen Sie die Datei aus, die Sie freigeben möchten und bestätigen Sie.</li>
  </ol>
  </TabItem>
</Tabs>

## Bilder und Videos teilen

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie eine Gruppe oder einen Privatchat, mit dem Sie ein Bild oder Vide teilen möchten.</li>
    <li>Klicken Sie auf das <b>Büroklammer-Symbol</b>.</li>
    <li>Wählen Sie das Bild/Video aus Ihrem Dateisystem aus.</li>
    <li>Klicken Sie auf <b>Ja</b>, um das Teilen zu bestätigen..</li>
  </ol>

:::info

Sie können Bilder oder Videos auch in den Chat ziehen, wo Sie sie verschicken möchten.

:::

:::info

Kopierte Bilder oder Videos können auch direkt in den Chat eingefügt werden, entweder durch das Rechtsklickmenü in der Nachrichtenzeile oder durch ein Tastenkürzel. Unter Firefox kann nur eine Datei gleichzeitig eingefügt werden.

:::

  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Öffnen Sie eine Gruppe oder einen Privatchat, mit dem Sie ein Bild oder Vido teilen möchten.</li>
    <li>Tippen Sie <b>Video</b>, <b>Galerie</b> oder <b>Kamera</b>.</li>
    <li>Wählen Sie ein Bild/Video oder machen Sie ein Bild/Video mit der Kamera.</li>
    <li>Optional: Tippen Sie auf <b>Schwarz</b> wenn Sie Teile des Bildes schwärzen möchten.</li>
    <li>Tippen Sie auf das <b>Häkchensymbol</b>.</li>
  </ol>
  </TabItem>
</Tabs>

* * *

## Dateien und Bilder herunterladen

Sie können Dateien und Bilder, die Sie in einem Chat erhalten haben, herunterladen.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie den Chat mit der Datei, die Sie herunterladen möchten.</li>
    <li>Klicken Sie auf die Datei, dann auf den Downloadpfeil in der oberen rechten Ecke.</li>
    <li>Bestätigen Sie die Benachrichtigung.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Öffnen Sie den Chat mit der Datei, die Sie herunterladen möchten.</li>
    <li>Klicken Sie auf die Datei, dann auf den Downloadpfeil in der oberen rechten Ecke.</li>
    <li>Bestätigen Sie die Benachrichtigung.</li>
  </ol>
  </TabItem>
</Tabs>

* * *

## Bildabschnitte schwärzen

<Tabs>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Folgen Sie den Schritten zum Teilen eines Bildes.</li>
    <li>Bevor Sie auf das <b>Häkchensymbol</b> tippen, können Sie die Farben schwarz oder weiß auswählen, indem Sie auf das kleine Maler Scroll-Symbol in der unteren rechten Ecke der App klicken, wenn Sie ein Bild ausgewählt haben.</li>
    <li>Tippe auf das <b>Checkmark-Symbol</b> nachdem du fertig bist.</li>
    </ol>
  </TabItem>
</Tabs>
