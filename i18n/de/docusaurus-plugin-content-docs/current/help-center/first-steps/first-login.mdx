## Allgemein

Benutzerkonten werden immer durch Ihre Organisation vergeben. Der Arbeitgeber wird Ihnen ein Willkommensblatt zur Verfügung stellen, das Ihnen einen schnellen und einfachen Zugang zu Famedly ermöglicht.

### Login

<Tabs groupId="operating-systems">
  <TabItem value="web" label="Web">
  <ol>
    <li>Geben Sie Ihre Organisations-URL in Ihrem Browser ein.</li>
    <li>Geben Sie <code>Benutzername</code> und <code>Passwort</code> ein.</li>
    <li>Stellen Sie einen Profilbild ein auf der Willkommensseite.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil" default>
  <ol>
    <li>Installieren Sie Famedly auf Ihrem Smartphone.</li>
    <li>Geben Sie die Organisations-URL ein.</li>
    <li>Geben Sie <code>Benutzername</code> und <code>Passwort</code> ein.</li>
    <li>Stellen Sie ein Profilbild ein auf der Willkommensseite.</li>
  </ol>
  </TabItem>
</Tabs>

### Chat-Backup aktivieren

Es ist sehr wichtig, das Chat-Backup zu aktivieren. Folgen Sie diesen Schritten, um das Chat-Backup zu aktivieren.

<Tabs groupId="operating-systems">
  <TabItem value="web" label="Web">
  <ol>
    <li>Klicken Sie auf Ihren Profilbild oder Ihre Initialien um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf <b>Chat-Backup.</b></li>
    <li>Klicken Sie auf <b>Chat-Backup aktivieren</b>.</li>
    <li>Heben Sie die Passphrase an einem sicheren Ort auf.</li>
    <li>Klicken Sie auf <b>Aktivieren</b>.</li>    
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil" default>
  <ol>
    <li>Tippen Sie auf Ihren Profilbild oder Ihre Initialien um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie auf <b>Chat-Backup.</b></li>
    <li><b>Tippen Sie auf Chat-Backup aktivieren</b>.</li>
    <li>Heben Sie die Passphrase an einem sicheren Ort auf.</li>
    <li>Tippen Sie auf <b>Aktivieren</b>.</li>
  </ol>
  </TabItem>
</Tabs>

:::info

**Wie kann ich eine Passphrase erstellen?**

If you have not yet created a chat backup and have not yet received a passphrase you can set one up the first time you log in to the web browser or mobile app.

**Wann brauche ich die Passphrase?**

Sie benötigen die Passphrase nur, wenn Sie auf allen Geräten abgemeldet sind und die Verifikation zwischen zwei Geräten deshalb nicht möglich ist. In diesem Fall muss die Passphrase eingegeben werden, sodass alle alten Nachrichten nach dem Login entschlüsselt werden.

**What happens if I have lost the passphrase and am logged out on all devices?**

Wenn Sie Ihre Passphrase nicht sicher aufgehoben haben oder sie nicht mehr finden können, und Sie von allen Geräten abgemeldet wurden, können Sie Ihr Konto nur noch zurücksetzen. Ihre Kontakte und Gruppen bleiben behalten, aber vorherige Nachrichten können nicht mehr entschlüsselt werden. 

:::

:::caution

Mit der Passphrase haben Sie Zugriff auf alle Ihre verschlüsselten Daten und vorherigen Chatverläufe. You need this passphrase to recover your chat history if you have been logged out on all devices.
Die Passphrase kann nicht wiederhergestellt werden, wenn Sie sie verloren haben!

:::
