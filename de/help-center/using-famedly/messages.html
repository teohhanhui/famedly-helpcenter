Nachrichten senden​

  • Desktop
  • Mobil
  1. Öffnen Sie den Gruppenchat oder Privatchat, an den Sie eine Nachricht senden möchten.
  2. Klicken Sie auf das Nachrichtenfeld.
  3. Geben Sie Ihre Nachricht ein und fügen Sie alle gewünschten Emoji, Erwähnungen oder Formatierungen hinzu.
  4. Drücken Sie Enter oder klicken Sie auf den Pfeil, um diese zu senden.
  1. Öffnen Sie den Gruppenchat oder Privatchat, an den Sie eine Nachricht senden möchten.
  2. Tippen Sie auf das Nachrichtenfeld.
  3. Geben Sie Ihre Nachricht ein und fügen Sie alle gewünschten Emoji, Erwähnungen oder Formatierungen hinzu.
  4. Tippen Sie auf den Pfeil, um sie zu senden.
tipp

Mit Shift + Eingabe können Sie in eine neue Zeile springen.

Neue Nachrichten lesen​

Wird ein Gruppenchat oder Privatchat Chat-Liste fett angezeigt, bedeutet das, dass es in der Unterhaltung ungelesene Nachrichten gibt. Klicken oder tippen Sie auf den fetten Konversationsnamen, um die ungelesenen Nachrichten anzuzeigen.

Weil sich gelegentlich Fehler einschleichen, ist es in Famedly möglich, Nachrichten zu bearbeiten und zu löschen.

Nachrichten bearbeiten​

Standardmäßig können alle Mitglieder ihre Nachrichten bearbeiten.

  • Desktop
  • Mobil
  1. Bewegen Sie den Cursor über die Nachricht, die Sie bearbeiten möchten.
  2. Klicken Sie auf das ✎ Bleistift-Symbol.
  3. Bearbeiten Sie Ihre Nachricht im Textfeld.
  4. Klicken Sie auf das →Senden Symbol.
  1. Tippen Sie auf die Nachricht, die Sie bearbeiten möchten.
  2. Tippen Sie Bearbeiten um Änderungen vorzunehmen.
  3. Bearbeiten Sie Ihre Nachricht im Textfeld.
  4. Tippen Sie auf das ✔️ Häkchen Symbol um den Prozess abzuschließen.

Nachrichten löschen​

Standardmäßig können alle Mitglieder ihre Nachrichten löschen.

info

Das Löschen einer Nachricht ist permanent, also überlegen Sie es sich vorher gut.

  • Desktop
  • Mobil
  1. Bewegen Sie den Mauszeiger über die Nachricht, die Sie bearbeiten möchten.
  2. Klicken Sie auf das Papierkorb-Symbol.
  3. Klicken Sie auf Ja, um zu bestätigen.
  1. Tippen Sie auf die Nachricht, die Sie löschen möchten.
  2. Tippen Sie auf Löschen, um die Nachricht zu löschen.
  3. Klicken Sie auf Ja, um zu bestätigen.

Auf Nachrichten antworten​

Mit der Antwortfunktion können Sie in Gruppenchat oder Privatchat auf eine bestimmte Nachricht antworten.

  • Desktop
  • Mobil
  1. Bewegen Sie den Mauszeiger über die Nachricht, die Sie beantworten möchten.
  2. Klicken Sie auf das Antwort-Symbol.
  3. Geben Sie Ihre Antwort in das Textfeld ein.
  4. Drücken Sie Enter oder klicken Sie auf den Pfeil, um die Antwort zu senden.
  1. Tippen Sie auf die Nachricht, die Sie beantworten möchten.
  2. Tippen Sie auf Antwort.
  3. Geben Sie Ihre Antwort in das Textfeld ein.
  4. Klicken Sie auf das Senden Symbol, um die Antwort zu senden.

Emojis zu Ihren Nachrichten hinzufügen​

Emojis sind eine Bilddarstellung von den textbasierten Emoticons. In Famedly sind Emojis sowohl unterhaltsam als auch praktisch für effizienteres Arbeiten: Verwenden Sie Emojis, um ihre Nachrichten zu verfeinern oder fügen Sie Emoji-Reaktionen einer Nachricht hinzu, um schnell darauf zu reagieren.

  • MacOS
  • Windows
  • iOS
  • Android
Benutzen Sie die Emoji-Tastatur um ein Emoji zu versenden. Bitte schauen Sie sich den Link unten an, um zu sehen, wie Sie die Emoji-Tastatur öffnen können.

Mehr Informationen

  1. Um in Windows ein Emoji hinzuzufügen, tippen Sie auf das Windows-Logo + . um die Emoji-Tastatur zu öffnen.
  1. Tippen Sie in der Eingabeleiste um die Tastatur zu öffnen.
  2. Tippen Sie auf das Emoji oder den Globus Knopf auf der Tastatur.
  3. Tippen Sie auf den gewünschten Emoji um ihn zur Nachricht hinzuzufügen.
  1. Benutzen Sie die Emoji Tastatur von Ihrem Gerät um ein Emoji einzufügen.

Mit einem Emoji auf eine Nachricht reagieren​

Oftmals kann eine weitere Nachricht durch eine Emoji-Reaktion ersetzt werden. Sie können sehen wer mit einem Emoji reagiert hat, indem Sie mit der Maus oder mit dem Finger die Emoji-Reaktion gedrückt halten.

Reaktionen hinzufügen​

  • Desktop
  • Mobil
  1. Klicken Sie auf die Nachricht, worauf Sie reagieren wollen.
  2. Wählen Sie einen Emoji aus den Vorschlägen oder klicken Sie auf das ＋ Plus-Symbol um ein anderes Emoji auszuwählen.
  1. Tippen Sie auf die Nachricht, worauf Sie reagieren wollen.
  2. Wählen Sie einen Emoji aus den Vorschlägen oder tippen Sie auf das ＋ Plus-Symbol um ein anderes Emoji auszuwählen.

Reaktionen entfernen​

Klicken oder tippen Sie einfach auf die Reaktion unter der Nachricht, um sie wieder zu entfernen.

info

Sie können nur Ihre eigenen Reaktionen löschen.

Jemanden in einer Nachricht erwähnen​

Indem Sie mit dem @ jemanden in einer Nachricht erwähnen, können Sie Personen direkt über etwas benachrichtigen, dass ihre Aufmerksamkeit in Famedly erfordert. Erfahren Sie, wie Sie Personen in Ihren Nachrichten erwähnen können.

  • Desktop
  • Mobil
  1. Schreiben Sie das @-Symbol, wenn Sie eine Nachricht schreiben.
  2. Geben Sie den Namen eines Persons ein oder wählen Sie eine Person aus der Liste. Sie können diesen Schritt wiederholen für jede Person, die Sie in einer Nachricht erwähnen wollen.
  3. Verschicken Sie die Nachricht.
  1. Schreiben Sie das @-Symbol in die Nachrichtenleiste.
  2. Geben Sie den Namen von der Person, die Sie erwähnen wollen ein oder wählen Sie sie aus der Liste aus. Sie können diesen Schritt für jede Person wiederholen, die Sie in Ihrer Nachricht erwähnen möchten.
  3. Senden Sie Ihre Nachricht.
info

Sie können nur Personen erwähnen die Mitglieder der Unterhaltung sind, sowohl in private Unterhaltungen als auch in Gruppenchats. Wenn Sie eine Person erwähnen wollen, die nicht Mitglied der Gruppe ist, muss diese Person zuerst der Gruppe hinzugefügt werden.

Nachrichten weiterleiten​

  • Desktop
  • Mobil
  1. Klicken Sie auf die Nachricht, die Sie weiterleiten möchten.
  2. Klicken Sie auf Nachricht weiterleiten
  3. Wählen Sie die Unterhaltung, in die Sie die Nachricht weiterleiten möchten.
  1. Tippen Sie auf die Nachricht, die Sie weiterleiten möchten.
  2. Tippen Sie auf Nachricht weiterleiten
  3. Wählen Sie die Unterhaltung, in die Sie die Nachricht weiterleiten möchten.

Nachrichteninhalt melden​

Die Nachricht wird dem Administrator Ihrer Organisation gemeldet.

  • Desktop
  • Mobil
  1. Klicken Sie auf die Nachricht, die Sie melden wollen.
  2. Klicken Sie auf Anstößigen Inhalt melden
  1. Halten Sie die Nachricht gedrückt mit der Maus oder mit Ihrem Finger.
  2. Tippen Sie auf Anstößigen Inhalt melden

Einsehen wer Nachrichten gelesen hat​

  • Desktop
  • Mobil
  1. Klicken Sie auf die Nachricht, von der Sie wissen wollen, wer sie gelesen hat.
  2. Klicken Sie auf Gelesen
  1. Halten Sie die Nachricht lange gedrückt, von der Sie wissen wollen, wer sie gelesen hat.
  2. Tippen Sie auf gelesen

Nachrichten formatieren​

Mit Hilfe der Formatierung können Sie Ihre Nachrichten in Famedly detaillierter und klarer gestalten. Sie können Markup-Sprache verwenden und den Text von ausgewählten Sonderzeichen umgeben.

Wenn Sie Famedly auf Ihrem Desktop oder mobil verwenden, können Sie Nachrichten mit diesen Befehlen formatieren.

FormatBeschreibung
FettText mit Sternen umgeben: Ihr Text
KursivText mit Unterstrichen umgeben: Ihr Text
DurchgestrichenText mit Tilden umgeben: ~Ihr Text~
CodeText mit Backticks umgeben: Ihr Text
ZitatFügen Sie eine spitze Klammer vor den Text: >Ihr Text
Code blockFügen Sie drei Backticks vor und nach dem Text hinzu: Ihr Text
Überschrift 1Fügen Sie eine Route ein und drücken Sie die Leertaste vor dem Text: # Ihr Text
Überschrift 2Fügen Sie zwei Routen ein und drücken Sie die Leertaste vor dem Text: ## Ihr Text
Überschrift 3Fügen Sie drei Routen ein und drücken Sie die Leertaste vor dem Text: ### Ihr Text
1. Nummerierte ListeSchreiben Sie 1 und einen Punkt und dann ein Leerzeichen vor dem Text: 1. Ihr Text.
- AufzählungslisteFügen Sie ein Sternchen und ein Leerzeichen vor Ihrem Text ein.
