## Log in

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Open Famedly web app.</li>
    <li>Enter your matrix servers URL in the input field <b>Login URL</b>.</li>
    <li>Click on <b>Next</b>.</li>
    <li>Enter your username in the input field <b>Username</b> and your password in the input field <b>Password</b>.</li>
    <li>Click on <b>Login</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Open Famedly app.</li>
    <li>Enter your matrix servers URL in the input field <b>Login URL</b>.</li>
    <li>Tap on “Next”.</li>
    <li>Enter your username in the input field <b>Username</b> and your password in the input field <b>Password</b>.</li>
    <li>Tap on “Login”.</li>
  </ol>
  </TabItem>
</Tabs>

## Global account settings on login

After login you will be directed to a page with different switches for <b>Privacy</b> settings (chat features) and for the <b>User directory</b>.  

You can find a description for the chat features here: [Privacy settings](../your-account/account-settings#privacy)  
A description for the user directory can be found here: [User directory](../your-account/account-settings#directory)

If you already activated the [chat backup](../first-steps/first-login#activate-chat-backup) you will be redirected to the verification process after clicking on <b>Continue</b>. 

## Verification of your chat backup 

Before you can read all your messages you need to verify your account. You can find all information for that on [QR-Verification](../using-famedly/qr-verification). 
There is also a fallback for comparing numbers if you for some reason can not scan QR-codes. The usage of that is also described on [QR-Verification](../using-famedly/qr-verification).
